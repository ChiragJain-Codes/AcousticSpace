# AcousticSpace

## Overview
AcousticSpace is an AI-based voice spoof detection system that classifies an uploaded audio file as either:
- Bonafide (Real Human Voice)
- Spoof (AI/Fake Voice)

---

## Tech Stack

- Python
- FastAPI
- Librosa
- NumPy
- Matplotlib
